# Modelo de madurez

## Nivel 0 — Ad hoc
Sin estándares consistentes.

## Nivel 1 — Base
Estructura, build, control de versiones y documentación mínima.

## Nivel 2 — Controlado
CI, pruebas, revisión, seguridad y despliegues reproducibles.

## Nivel 3 — Operable
Observabilidad, rollback, SLO, runbooks y respuesta a incidentes.

## Nivel 4 — Optimizado
Métricas de flujo, automatización avanzada, supply chain segura y mejora continua.

El objetivo no es alcanzar el máximo nivel en todos los proyectos, sino el nivel proporcional a su riesgo y vida útil.
