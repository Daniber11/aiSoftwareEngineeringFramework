# Estándar de observabilidad

- Logs estructurados con nivel, timestamp, servicio, versión y correlation ID.
- Métricas RED para servicios y USE para recursos cuando aplique.
- Trazas para flujos distribuidos críticos.
- Health checks diferenciados.
- Dashboards centrados en objetivos.
- Alertas accionables con propietario y runbook.
- Prohibido registrar secretos, credenciales o datos personales innecesarios.
