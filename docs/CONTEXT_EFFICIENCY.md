# Eficiencia de contexto para IA

## Índices antes que exploración

- `FRAMEWORK.yaml` descubre la configuración.
- `MODULES.md` localiza responsabilidades.
- `CURRENT_CONTEXT.md` comunica el foco.
- ADR evita rediscutir decisiones.
- Plantillas estructuran solicitudes.

## Sesiones

Al iniciar:
- Leer solo documentos de entrada y módulos afectados.

Durante:
- Mantener resúmenes breves.
- Trabajar por incrementos verticales.
- Reutilizar comandos de validación.

Al cerrar:
- Actualizar contexto activo.
- Mover conocimiento estable a su documento.
- Eliminar información temporal obsoleta.
