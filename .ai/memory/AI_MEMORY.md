# Memoria técnica estable

Guarda únicamente conocimiento duradero que reduzca futuras exploraciones:

- Convenciones no obvias.
- Restricciones permanentes.
- Patrones aprobados.
- Antipatrones prohibidos.
- Decisiones operativas recurrentes.
- Comandos de validación confiables.

No almacenar aquí el estado temporal de una tarea ni duplicar ADR.
