# Checklist de tarea para IA

- [ ] Leí manifiesto, contexto y política.
- [ ] Identifiqué el alcance mínimo.
- [ ] No hice suposiciones críticas ocultas.
- [ ] No modifiqué áreas no relacionadas.
- [ ] Agregué o actualicé pruebas.
- [ ] Verifiqué seguridad y datos.
- [ ] Actualicé documentación.
- [ ] Reporté validaciones reales.
- [ ] Registré ADR si era obligatorio.
- [ ] Actualicé contexto activo.
