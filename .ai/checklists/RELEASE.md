# Checklist de release

- [ ] Build reproducible.
- [ ] Pruebas requeridas aprobadas.
- [ ] Escaneos de seguridad aprobados.
- [ ] Migraciones verificadas.
- [ ] Compatibilidad revisada.
- [ ] Rendimiento crítico validado.
- [ ] Logs, métricas y alertas revisados.
- [ ] Runbook actualizado.
- [ ] Changelog y versión actualizados.
- [ ] Rollback probado o documentado.
- [ ] Evidencia de aprobación disponible.
