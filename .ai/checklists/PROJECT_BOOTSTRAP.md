# Checklist de inicio

- [ ] Propósito, alcance y actores definidos.
- [ ] Riesgo y atributos de calidad definidos.
- [ ] Arquitectura inicial documentada.
- [ ] Estrategia de datos y migraciones definida.
- [ ] Seguridad y amenazas iniciales revisadas.
- [ ] Estrategia de pruebas definida.
- [ ] CI con quality gates activo.
- [ ] Secret scanning y dependency scanning activos.
- [ ] Entornos reproducibles.
- [ ] Observabilidad mínima.
- [ ] Estrategia de despliegue y rollback.
- [ ] Protección de ramas y revisión.
- [ ] Documentos de IA inicializados.
- [ ] ADR inicial aprobado.
