# Atributos de calidad

Define objetivos medibles para:

- Disponibilidad.
- Latencia.
- Rendimiento.
- Escalabilidad.
- Seguridad.
- Privacidad.
- Resiliencia.
- Recuperación.
- Mantenibilidad.
- Accesibilidad.
- Compatibilidad.
- Costos.

Cada objetivo debe incluir métrica, umbral, escenario y método de validación.
